# Chetas — Desktop AI Assistant

Chetas is a Windows desktop AI assistant that provides:

- Voice activation
- Text chat
- System automation
- Short-term memory
- Lightweight PyQt5 GUI

This repository contains the UI, core orchestration, audio I/O helpers, and utility features.

---

# Features

- Voice wake-word support
- Natural speech responses (TTS)
- Command parsing
- Open apps
- Control volume
- Lock / shutdown system
- Memory storage and recall
- Modular project structure

---

# Tech Stack

- Python 3.10+
- PyQt5
- Ollama
- Windows SAPI (fallback TTS)
